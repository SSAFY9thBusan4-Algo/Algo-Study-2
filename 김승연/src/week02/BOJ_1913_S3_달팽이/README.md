<h1> 문제 접근 </h1>

일단 예시로 나와있는 형태로 2차원 배열에 값을 저장하여 출력하면 되지 않을까 생각했다.

그런데 그냥 봐서는 규칙이 잘 보이지 않았기 때문에 다음과 같이 row와 col의 인덱스 변화를 적어보았다.


 * 5X5 예시(n = 5)
 
 
 * 초기값
 * r = n / 2;
 * c = n / 2;
 
 
 * r -= 1;
 * c += 1;
 * r += 2;
 * c -= 2;
 
 
 * r -= 3;
 * c += 3;
 * r += 4;
 * c -= 4;
 
 
예시를 적어보자 규칙이 보였다! 그래서 그걸 그대로 구현했다.

여기서 1, 1, 2, 2, ..와 같이 점점 증가하는 수는 코드의 val 변수로 나타냈고,

부호의 변화는 sign으로 나타냈다.
