<h3> 문제 풀이 방법 </h3>

- 강의에서 배운 코드를 쓰니 바로 문제가 풀렸다.
- 0부터 N까지 서로소 집합 N+1 개를 만든다.
- 명령을 입력받는데 0 a b 이면 a가 대표자인 집합과 b가 대표자이 집합을 합친다.
- 1 a b 이면 a와 b가 같은 집합에 있을 경우 YES, 아니면 NO를 출력한다.
