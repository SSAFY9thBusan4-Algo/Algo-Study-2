<h3> 문제 풀이 방법 </h3>

- 인접 리스트를 생성한 후, 인접한 애들 탐색 (인접 리스트 생성을 위해 노드 클래스 생성)
- 최단 경로를 구하는 문제이고, 모든 도로의 거리가 1로 고정되어 있기 때문에 BFS 탐색으로 풀었다.
- X 노드부터의 거리를 저장하는 배열 distance를 만들어 거리를 저장한 후,  
  거리가 K인 정점이 존재하면 해당 정점 출력, K인 정점이 없다면 -1 출력
